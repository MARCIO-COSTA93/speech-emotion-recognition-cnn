\# Speech Emotion Recognition using CNN



\##  Overview

This project implements a Speech Emotion Recognition (SER) system using a Convolutional Neural Network (CNN). The goal is to classify human emotions from audio signals.



\##  Technologies

\- Python

\- TensorFlow / Keras

\- Librosa

\- NumPy

\- Matplotlib



\##  Dataset

RAVDESS (Ryerson Audio-Visual Database of Emotional Speech and Song)



\##  Methodology

\- Audio preprocessing

\- Feature extraction (MFCC)

\- CNN model training

\- Emotion classification
